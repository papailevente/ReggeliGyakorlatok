# Release Notes - v1.2.0 (MorningRoutine)

## 🚀 Key Updates
*   **Weekly Workout Planner**: You can now define different exercise sets for each day of the week! 
*   **Day Selector**: A new navigation bar on the workout screen allows you to easily switch between Monday and Sunday to manage your daily routines.
*   **Dynamic Loading**: The app automatically detects the current day and loads the corresponding exercises.
*   **Smart Database**: Upgraded the database to version 3 to support day-specific exercise definitions while preserving your settings.

## 🛠 Bug Fixes & Improvements
*   **Version Bump**: Updated Version Code to `14` and Version Name to `1.2.0`.
*   **Localization**: Added translations for day names in Hungarian, German, and English.
*   **Initialization**: Existing exercises are preserved as "Every Day" (0) or assigned to their current context.
