# Release Notes - v1.2.1 (MorningRoutine)

## 🚀 Key Updates
*   **Enhanced Exercise Editor**: When editing an exercise, you can now directly select which day of the week it should be active on.
*   **Day Selection UI**: Added a horizontal scrollable list of days in the edit dialog with visual feedback (darker background and yellow highlight for selected days).
*   **Flexible Scheduling**: Easily move exercises between days or assign them to "Every Day" (∞).

## 🛠 Bug Fixes & Improvements
*   **Version Bump**: Updated Version Code to `15` and Version Name to `1.2.1`.
*   **UI Polish**: Improved dialog layouts for better readability on smaller screens.
