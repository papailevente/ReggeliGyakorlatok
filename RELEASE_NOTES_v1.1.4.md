# Release Notes - v1.1.4 (MorningRoutine)

## 🚀 Key Updates
*   **Workout Auto-Start**: Tapping the "Training" button in the bottom navigation bar now automatically starts the total workout timer.
*   **History Saving Fixed**: Resolved an issue where workout results were not being saved to the history database.
*   **Update Migration Choice**: When the app is updated, a new dialog asks if you want to clear your old history for a fresh start or keep your existing data.

## 🛠 Bug Fixes & Improvements
*   **Database Reliability**: Improved the way workout records are persisted.
*   **Localized Prompts**: Added Hungarian, German, and English support for the new update migration messages.
*   **Version Bump**: Updated Version Code to `10` and Version Name to `1.1.4`.
