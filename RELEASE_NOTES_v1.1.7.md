# Release Notes - v1.1.7 (MorningRoutine)

## 🚀 Key Updates
*   **Manual Workout Completion**: The app no longer finishes the workout automatically when the last set is done. Instead, it waits for you to manually tap the flashing "Finish Workout" button, giving you time to review your stats.
*   **Visual Feedback**: The "Finish Workout" button correctly triggers its red/blue flashing animation as soon as all exercises and sets are completed.

## 🛠 Bug Fixes & Improvements
*   **Version Bump**: Updated Version Code to `13` and Version Name to `1.1.7`.
*   **Flow Control**: Improved the logic for handling the transition from the last exercise to the workout completion state.
