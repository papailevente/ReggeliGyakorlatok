# Release Notes - v1.3.0 (MorningRoutine)

## 🚀 Key Updates
*   **Exercise Groups**: You can now organize your exercises into custom groups.
*   **Normal & Extra Modes**: 
    *   **Normal Mode**: Shows all exercises scheduled for the current day as before.
    *   **Extra Mode**: Allows you to pick a specific Exercise Group to focus on.
*   **Group Management**: Added a new manager to create, edit, and assign groups to specific days.
*   **Enhanced Exercise Editor**: Assign exercises to specific groups or keep them in the general pool.

## 🛠 Bug Fixes & Improvements
*   **Version Bump**: Updated Version Code to `16` and Version Name to `1.3.0`.
*   **Mode Selector**: Added a clean mode selection screen after the splash screen.
*   **UI Polish**: Optimized layout for group and exercise management.
