# Release Notes - v1.1.2 (Morning Routin)

## 🚀 Key Updates
*   **App Renaming**: The application has been officially renamed from "Reggeli Rutin" to **"Morning Routin"** for a more global appeal.
*   **New Brand Identity**: Integrated a high-resolution (512x512) application icon based on the splash screen design for a sharper, more professional look on home screens.
*   **Enhanced Splash Screen**: Improved the launch experience by optimizing the transition between the system splash and the custom branded splash screen.

## 🛠 Bug Fixes & Improvements
*   **Database Stability**: Fixed a critical `IllegalStateException` related to Room database migrations (Version 1 -> 2). Existing user data preservation is now prioritized with a safety fallback mechanism.
*   **Version Bump**: Updated Version Code to `8` and Version Name to `1.1.2`.
*   **UI Tweaks**: Refined the adaptive icon layers to ensure perfect centering and compatibility with Android 12+ "Themed Icons".

*   **Build Date**: May 2026
*   **Target SDK**: 35
*   **Min SDK**: 24
