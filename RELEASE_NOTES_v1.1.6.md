# Release Notes - v1.1.6 (MorningRoutine)

## 🚀 Key Updates
*   **Circuit Training Mode**: The app now automatically switches to the next incomplete exercise after **every single set**. 
*   **Circular Progression**: When the end of the list is reached, it automatically wraps around to the beginning to find any remaining sets.
*   **Smart Navigation**: The focus automatically shifts to keep you moving through your routine without manual intervention.

## 🛠 Bug Fixes & Improvements
*   **Version Bump**: Updated Version Code to `12` and Version Name to `1.1.6`.
*   **Flow Optimization**: Refined the logic for detecting finished workouts and switching focus.
